# AI Prompts

A personal list of AI Prompts that I am using.

## Files and Folders

- `.github\prompts\*.prompt.md` - VS Code Github Copilot [Prompt-files]( https://code.visualstudio.com/docs/copilot/customization/prompt-files)
- `tasks\` - A folder to hold the outputs from these prompts
- `tasks\done` - A folder to hold the compleated tasks.
